## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do **Checkpoint 02** da disciplina de *Mobile Application Development*.

O objetivo do aplicativo é permitir o cadastro de informações de um aluno por meio de um formulário, contendo os campos nome, RM, telefone e CPF. Após o envio, os dados são exibidos em uma tela de perfil.

---

## 🚀 Funcionalidades

* Cadastro de aluno
* Validação de campos obrigatórios
* Máscara de input para telefone e CPF
* Persistência de dados com AsyncStorage
* Autopreenchimento ao abrir o aplicativo
* Navegação entre telas (Cadastro → Perfil)

---

## 🧠 Tecnologias Utilizadas

* React Native
* JavaScript
* Expo
* React Navigation
* AsyncStorage

---

## 📚 Conceitos Aplicados

* useState para gerenciamento de estado
* useEffect para carregamento de dados
* Navegação entre telas
* Armazenamento local de dados
* Componentes do React Native
* StyleSheet para estilização
